# Mimos de Luz - Landing Page

Landing page para a Mimos de Luz, loja de presentes exclusivos e kits especiais feitos com amor e carinho.

## Tecnologias

- React 18
- Vite
- Tailwind CSS
- Framer Motion
- Lucide React (ícones)

## Instalação

```bash
npm install
```

## Desenvolvimento

```bash
npm run dev
```
